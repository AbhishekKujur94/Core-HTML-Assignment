List of Tags

  1. h1,....,h6: Defines headings of different levels, with h1 being the most important and h6 being the least important.
  2. p: Defines a paragraph of text.
  3. div: Defines a container for HTML elements.
  4. table: Defines a table with rows and columns.
  5. a: Creates a hyperlink that links to another webpage.
  6. img: Embeds an image into the HTML document.
  7. ol: Defines an ordered list.
  8. ul: Defines an unordered list.
  9. li: Defines each item within the list.
